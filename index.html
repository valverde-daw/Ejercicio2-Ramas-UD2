Resumen de comandos Git

  • git init: Inicializa un repositorio Git.
  • git add: Añade archivos al área de preparación.
  • git commit: Guarda los cambios en el repositorio local.
  • git push: Envía los cambios al repositorio remoto.
  • git pull: Descarga y fusiona cambios del remoto.
  • git branch: Gestiona ramas en el repositorio.
  • git checkout: Cambia entre ramas o commits.
  • git merge: Fusiona ramas.
  • git rebase: Integra cambios desde otra rama.
